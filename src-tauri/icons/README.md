# Icons

This directory contains application icons for different platforms.

Required icon files:
- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

For now, placeholder files are needed. You can generate proper icons using tools like:
- https://icon.kitchen/
- https://www.favicon-generator.org/
